# chess-mate-parttern

## 📁 프로젝트 목표

- `chess.js`를 정답지로 활용하는 `TDD` 환경을 구축하여 퍼즐의 정답을 판별하는 모델을 구현
- 퍼즐 데이터, 데이터 관리, 게임 로직 등의 클래스가 책임을 분리
- `MVC 패턴` 적용 및 `React`를 `View`로 활용하여 UI 체스 게임을 렌더링

## 핵심 기능 요구사항

1. 메인 화면(패턴 목록)

- 사용자는 풀고 싶은 메이트 패턴을 선택하기 위해, 메인 화면에서 퍼즐 목록을 볼 수 있다.
- 목록에는 각 퍼즐의 이름과 난이도(또는 해결 여부)가 표시된다.

2. 퍼즐 화면

- 패턴을 선택하면, 해당 패턴의 시작 FEN(체스판 상태)을 React 컴포넌트를 이용해 웹 화면에 렌더링한다.
- 퍼즐의 이름과 퍼즐의 조건 등의 설명을 표시한다.

3. 기물 이동

- 사용자는 퍼즐을 풀기 위해, 백의 차례일 때 백색 기물을, 흑의 차례일 때 흑색 기물을 클릭하여 이동시킬 수 있다.
- 기물을 클릭하면 이동 가능한 칸들이 하이라이트되어야 한다.

4. 결과 피드백

- Model의 판별 결과에 따라 View는 "정답입니다!" 또는 "틀렸습니다. 다시 시도하세요." 메시지를 출력한다.
- 정답이 아닐 경우, 기물은 원래 자리로 돌아간다.
- (고려사항) 기물 힌트 버튼 존재, 힌트는 기물을 하늘색으로 표시

## 📋 요구사항 명세서

1. 메인화면에서 여러 메이트 패턴 중 하나를 선택한다.
2. 패턴 선택 시 패턴에 맞는 체스 보드가 나타나며, 퀴즈 조건을 보여준다.
3. 사용자가 옳은 수를 두었을 때 성공이며, 옳지 않은 수를 두었을 때는 실패이며, 처음 체스 보드로 돌아간다.
4. 사용자가 퀴즈를 풀면 추가적인 패턴 설명을 보여주고, 처음 화면으로 돌아간다.
